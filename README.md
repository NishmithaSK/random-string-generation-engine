# Random String Generation Engine

A responsive React web application built with Vite and Tailwind CSS v4 designed to generate customizable alphanumeric strings and tokens dynamically.

## Features
* **Custom Character Composition:** Toggle parameters to include or exclude uppercase letters, lowercase letters, numeric digits, and special symbols.
* **Variable Length Controls:** Seamless slider interface allowing precise adjustment of string outputs ranging from 4 to 32 characters.
* **Instant UI Synchronization:** Fully reactive design that automatically updates the output string the moment any parameter is modified.
* **One-Click Clipboard Integration:** Built-in copy function featuring an asynchronous status indicator (`Copy` → `Copied!`).

## Tech Stack
* React 19
* Vite
* Tailwind CSS v4
